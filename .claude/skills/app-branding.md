# Color Tokens

Background:     #f7eacb  (Warm Cream)
Primary CTA:    #008c4c  (Emerald Green)
Secondary UI:   #2f6b3f  (Forest Green)
Accent:         #ffd23f  (Amber Yellow)
Text / Dark:    #1b231d  (Dark Charcoal)


Use this exact color system throughout the UI:

- Background: #f7eacb (warm cream) — use for page and card backgrounds
- Primary CTA (buttons, links, active states): #008c4c (emerald green)
- Secondary UI (navbar, sidebar, section blocks, hover states): #2f6b3f (forest green)
- Accent (badges, tags, highlights, promotional elements): #ffd23f (amber yellow) — use sparingly
- Text and dark surfaces: #1b231d (dark charcoal)

Color ratio to follow: 60% cream background, 30% greens split between the two, 5% amber accent, 5% dark charcoal for text. Do not introduce any colors outside this palette. Maintain WCAG AA contrast compliance, especially on CTA buttons. The overall feel should be earthy, premium, and nature-forward.
